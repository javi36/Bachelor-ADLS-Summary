# 5. Reaktionen & Mechanismen

!!! abstract "Modulinhalt"
    Aus dem Handout *Reaktionen und Mechanismen*. Reaktivität folgt aus den Bindungen: **Alkane** sind unpolar und reaktionsträge (nur starke σ-Bindungen, keine funktionelle Gruppe), **Alkene/Alkine** sind reaktiver (zugängliche, polarisierbare π-Bindung), **Halogenalkane** sind durch die polare C–X-Bindung reaktiv.

## 1. Radikalische Halogenierung (Alkane)

Bedingungen: **hν (UV-Licht)** oder **T > 300 °C** → homolytischer Bindungsbruch.

1. **Initiation (Kettenstart)**: Cl₂ → 2 Cl•
2. **Propagation (Kettenfortpflanzung)**: CH₄ + Cl• → CH₃• + HCl; CH₃• + Cl₂ → CH₃Cl + Cl•
3. **Termination (Kettenabbruch)**: zwei Radikale rekombinieren (z. B. CH₃• + Cl• → CH₃Cl)

Die Reaktion lässt sich **nicht gezielt anhalten** → Produktgemisch: CH₄ → CH₃Cl → CH₂Cl₂ → CHCl₃ → CCl₄.

## 2. Radikalstabilität & Selektivität

Stabilität der Radikale: **tertiär > sekundär > primär** (Hyperkonjugation, induktiver Effekt, ggf. Resonanz). Das stabilste Radikal entsteht bevorzugt.

| Halogen | Reaktivität | Selektivität |
|---------|------------|-------------|
| F₂ | sehr hoch | sehr gering |
| Cl₂ | hoch | gering |
| Br₂ | mittel | hoch (tertiäre C bevorzugt) |
| I₂ | gering (endotherm) | sehr hoch |

**Relative Substitutionsgeschwindigkeit pro H-Atom** (25 °C):

| | primär | sekundär | tertiär |
|---|--------|----------|---------|
| Fluorierung | 1 | 1.2 | 1.4 |
| Chlorierung | 1 | 4 | 5 |

→ **Brom ist selektiver als Chlor**: weniger reaktiv ⇒ mehr Zeit, das stabilste Radikal zu bilden.

## 3. Reaktionen der Alkene

Reaktionen erfolgen **immer an der Doppelbindung**. Räumliche Begriffe:

- **syn-Addition**: beide Gruppen von **derselben** Seite.
- **anti-Addition**: Gruppen von **gegenüberliegenden** Seiten.

| Reaktion | Reagenz | Mechanismus / Produkt |
|----------|---------|------------------------|
| **Hydrierung** | H₂, Kat. (Pd/Pt/Ni) | **syn**-Addition → Alkan (techn.: Fetthärtung von Ölen) |
| **Halogenierung** | X₂ | **anti**-Addition über **zyklisches Halonium­ion** → 1,2-Dihalogenalkan |
| **Halohydrin-Bildung** | X₂ in H₂O | H₂O greift Haloniumion an → anti-Halogenalkohol (+ HX) |
| **Hydrohalogenierung** | HX | el. Addition, **Markovnikov**, über Carbokation → Halogenalkan |
| **Hydratation** | H₂O / H⁺ | Markovnikov → Alkohol |

### Halogenierung — warum *anti*?
Br₂ wird durch die Doppelbindung polarisiert und heterolytisch gespalten → **Bromoniumion** (zyklisch). Das Br⁻ greift von der **Rückseite** an (Vorderseite blockiert) → strikte anti-Addition. Carbokationen scheiden als Zwischenstufe aus, da sie keine stereochemische Kontrolle erlauben.

### Elektrophile Addition von HX — Markovnikov
HX wird heterolytisch gespalten → **Carbokation** + X⁻; danach C–X-Bindung. **Regel von Markovnikov**: H⁺ bindet an das C-Atom mit den **meisten H-Atomen**, weil so das **stabilere Carbokation** entsteht. Carbokation-Stabilität: **tertiär > sekundär > primär** (Hyperkonjugation, induktiver Effekt, Resonanz).

## 4. HX-Eliminierung (Alkene aus Halogenalkanen)

Rückreaktion zur HX-Addition: H und X werden abgespalten → Doppelbindung.

| | E1 | E2 |
|---|----|----|
| Verlauf | zweistufig über **Carbokation** (erst X⁻ ab, dann Deprotonierung) | **konzertiert** (X⁻-Abgang und Deprotonierung gleichzeitig) |

## 5. Nukleophile Substitution (Halogenalkane)

Polare C–X-Bindung: C ist δ⁺, X ist δ⁻. Ein **Nukleophil** (freies Elektronenpaar: OH⁻, RO⁻, CN⁻, NH₃, Hal⁻) ersetzt das Halogen — wichtige Synthese z. B. für Alkohole: `CH₃Br + OH⁻ → CH₃OH + Br⁻`.

| | SN2 | SN1 |
|---|-----|-----|
| Schritte | **1** (konzertiert) | **2** (über Carbokation) |
| Kinetik | v = k·[Nu]·[Edukt] (2. Ordnung) | v = k·[Edukt] (1. Ordnung) |
| Stereochemie | **Inversion** (Walden-Umkehr) | **Racemisierung** (planares Carbokation) |
| Begünstigt durch | primäre C, gutes Nu | tertiäre C, stabiles Carbokation |

## 6. Wichtige Reaktionstypen — Übersicht

| Typ | Kürzel | Beispiel |
|-----|--------|---------|
| Substitution radikalisch | SR | Halogenierung von Alkanen |
| Substitution elektrophil | SE | Nitrierung von Benzol |
