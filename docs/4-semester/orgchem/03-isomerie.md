# 3. Isomerie

!!! abstract "Modulinhalt"
    Aus dem Handout *Nomenklatur und Stereochemie*. Die Zahl der Isomere wächst rasant: Butan 2, Hexan 5, Decan 75 Konstitutionsisomere. Räumliche Anordnung kann Eigenschaften drastisch ändern (siehe Beispiele unten).

```
Isomerie
├── Konstitutionsisomerie (gleiche Summenformel, andere Verknüpfung)
└── Stereoisomerie (gleiche Verknüpfung, andere Raumanordnung)
    ├── Konfigurationsisomerie (Bindungsbruch nötig)
    │   ├── Enantiomerie (Bild/Spiegelbild, chirales C)
    │   └── Diastereomerie (cis/trans, E/Z; kein Spiegelbild)
    └── Konformationsisomerie (Rotation, kein Bindungsbruch)
```

## 1. Chiralität

- **Chiral**: Molekül und sein Spiegelbild **nicht zur Deckung** bringbar → die beiden sind **Enantiomere**.
- **Achiral**: Molekül ist mit seinem Spiegelbild identisch.
- Ursache: ein **asymmetrisches C-Atom** = sp³-C mit **4 verschiedenen** Substituenten (mit * markiert).

## 2. R/S-Nomenklatur (CIP, Cahn–Ingold–Prelog 1951)

1. Die 4 direkt gebundenen Atome nach **Ordnungszahl** priorisieren (① höchste … ④ niedrigste). Freie Elektronenpaare = niedrigste Priorität.
2. Bei Gleichstand: nächste „Sphäre" betrachten — höhere Priorität hat das Atom, das an Atome höherer Ordnungszahl/Masse gebunden ist (notfalls 3. Sphäre).
3. Substituent **④ nach hinten** orientieren, dann ① → ② → ③ ablesen:
    - **R** (rectus): im **Uhrzeigersinn**
    - **S** (sinister): **gegen** den Uhrzeigersinn
4. Doppel-/Dreifachbindungen: das gebundene Atom wird so gezählt, als wäre es **mehrfach** vorhanden (Duplikatatome).

## 3. E/Z-Isomerie (Doppelbindungen)

Weil Rotation um die Doppelbindung gesperrt ist (π-Bindung), sind cis/trans-artige Isomere fixiert:

- **Z** (zusammen): höher priorisierte Substituenten auf **gleicher** Seite.
- **E** (entgegen): höher priorisierte Substituenten auf **gegenüberliegenden** Seiten.

Eine Isomerisierung E ⇌ Z erfordert immer einen **Bruch der π-Bindung** (nicht der σ-Bindung).

## 4. cis/trans an Ringen

An Cycloalkanen mit Substituenten an verschiedenen C-Atomen: **cis** (gleiche Seite der Ringebene) bzw. **trans** (Gegenseiten) → Diastereomere.

## 5. Warum Stereochemie zählt — Beispiele

| Verbindung | Enantiomer A | Enantiomer B |
|------------|-------------|-------------|
| **Carvon** | (R): Pfefferminz | (S): Kümmel |
| **Thalidomid** (Contergan) | (R): beruhigend | (S): embryotoxisch |
| **Chinin / Chinidin** | fiebersenkend | wirkt aufs Herz |

→ Gleiche Summenformel, gleiche Bindungen — völlig unterschiedliche (Bio-)Wirkung.
